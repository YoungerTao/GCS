# GCS 安装说明（简化版）

## 一句话
**双击 `GCS-智能安装.bat` 一键完成所有安装**

## 完整步骤

### 第1步：安装
- 双击仓库根目录的 **`GCS-智能安装.bat`**
- 等待安装完成（首次 1-3 分钟）
- 看到 "✓ 安装成功" 提示

### 第2步：启动
- 双击桌面上的 **`GCS`** 图标
- 或双击仓库根目录的 **`GCS.cmd`**

## 系统要求
- Windows 10 / 11
- 网络连接（首次安装下载 Python）
- 至少 1 GB 磁盘空间

## 常见问题

**Q: 安装很慢？**
A: 首次安装需要下载 Python 和依赖包（100-300 MB），取决于网络速度。

**Q: 出现错误？**
A: 查看 `WINDOWS-INSTALL.md` 中的故障排查部分。

**Q: 想开机自启？**
A: 在安装脚本最后会提示创建开机启动快捷方式。

## 文件说明

| 文件 | 用途 | 用户需要知道 |
|------|------|-----------|
| `GCS-智能安装.bat` | **安装** | ✓ 必须双击一次 |
| `GCS.cmd` | **启动** | ✓ 日常双击启动 |
| `GCS.lnk`（桌面） | **快捷方式** | ✓ 由安装脚本自动创建 |
| `WINDOWS-INSTALL.md` | 详细文档 | 仅遇到问题时查看 |

## 更多帮助

- 详细文档：参考 `WINDOWS-INSTALL.md`
- 环境检查：运行 `powershell -ExecutionPolicy Bypass -File tools\check-windows-env.ps1`
- 开发者模式：见 `.claude/dev/gcs-dev.bat`
