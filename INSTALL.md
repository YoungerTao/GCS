# GCS 快速安装指南

## ⚡ 3 步启动

### 第 1 步：安装（仅需一次）
```
双击：GCS-智能安装.bat
等待：完成（首次 1-3 分钟）
```

### 第 2 步：启动（日常使用）
```
双击：桌面 GCS 图标
或：GCS.cmd
```

### 第 3 步：使用
```
Web UI 自动打开
开始使用 GCS
```

---

## 📋 系统要求

- **Windows**: 10 / 11
- **网络**: 用于首次下载 Python
- **磁盘**: ≥ 1 GB

## ⚙️ 可选优化

启用**开机加速**（快速启动）：
- 安装时选择"启用开机启动"
- 之后每次启动 < 2 秒（而不是 10-30 秒）

## ❓ 常见问题

**Q: 安装很慢？**
A: 首次需要下载 Python（100+ MB），取决于网络速度。

**Q: 出现错误？**
A: 查看 `docs/WINDOWS-INSTALL-DETAILED.md` 中的故障排查。

**Q: 想了解更多细节？**
A: 查看 `docs/FILE-CALL-FLOW.md` 了解文件调用流程。
