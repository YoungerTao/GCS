# 地形跟随测绘

## 概述

在「添加航测区域」中勾选 **用地形跟随规划航线** 后，航测航点使用 `MAV_FRAME_GLOBAL_TERRAIN_ALT`（frame 10），高度为相对地形的 AGL。规划依赖本地瓦片服务 `:8768` 的 SRTM 高程缓存。

## 用户流程

1. 画测区多边形（≥3 顶点）
2. 勾选地形跟随；可选自动切块、预取地形
3. 点击 **自动规划** → 切块、航向优化、剖面校核
4. 查看预览 issues 与可选 LLM 分析
5. **确认生成** 写入 `surveyBlocks` 与任务航点
6. 上传前 `MissionValidator` 校核；飞控需 `TERRAIN_ENABLE=1`

## 关键模块

| 模块 | 职责 |
|------|------|
| `tools/map-tiles/terrain_elevation.py` | SRTM 缓存、采样、MAVLink 网格编码 |
| `JS/services/terrain-service.js` | 前端高程 API 客户端 |
| `JS/mission/terrain-survey-planner.js` | 自动规划编排 |
| `JS/mission/terrain-profile-validator.js` | 爬升率 / DEM / frame 校核 |
| `JS/services/terrain-mavlink.js` | `TERRAIN_REQUEST` → `TERRAIN_DATA` |

## 设置项（`DEFAULT_MISSION_SETTINGS`）

- `useTerrainFollowing` — 主开关
- `terrainAgMarginM` — 安全裕度
- `terrainAutoPartition` / `terrainMaxReliefM` — 自动切块
- `terrainMaxClimbRateMps` / `terrainCruiseSpeedMps` — 剖面校核
- `terrainPrefetchOnDraw` — 画完测区自动预取

## LLM 顾问（可选）

`localStorage` 键 `gcs.terrainAdvisor`：`{ "enabled": true, "apiKey": "...", "baseUrl": "...", "model": "..." }`  
仅输出自然语言建议，不生成航点坐标。
