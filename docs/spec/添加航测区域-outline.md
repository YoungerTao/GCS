# 添加航测区域（XMind 规格摘要）

- **添加航测区域**
  - **判定机型**
    - **多旋翼**：航点1 TAKEOFF 30m；连机锁定；引导点2–3 默认100m；之后可手动或接测绘航点
    - **固定翼**：航点1 TAKEOFF；航点2 起飞引导 50m；坡度>20° 红线否则黄线；>200m 加 LOITER_TO_ALT；航点4 300m
    - **垂起固定翼**：航点1 VTOL_TAKEOFF 40m；其余同固定翼
  - **绘制区域**：先画多边形 → **确认生成**测绘航线
  - **返航**：可选 RTL
  - **保存航线**：`.waypoint`（ArduPilot / MP）
  - **写入 / 读取飞控任务**（MAVLink）
