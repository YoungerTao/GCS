# 添加航测区域（XMind 规格摘要）

- **添加航测区域**
  - **判定机型**
    - **多旋翼**：航点1 TAKEOFF 30m；连机锁定；引导点2–3 默认100m；之后可手动或接测绘航点
    - **固定翼**：起飞 → 引导 → 盘旋(>200m) → 引导（均为普通航点命名，引导即 NAV_WAYPOINT）
    - **垂起**：垂直起飞 → 引导 → 盘旋(>200m) → 引导；不插入 DO_VTOL_TRANSITION（自动模式飞控自行过渡固定翼）
  - **绘制区域**：先画多边形 → **确认生成**测绘航线
  - **返航**：可选 RTL
  - **保存航线**：`.waypoint`（ArduPilot / MP）
  - **写入 / 读取飞控任务**（MAVLink）
